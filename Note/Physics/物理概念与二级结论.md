## 同步卫星
+ 同步轨道有无数个，只要满足周期相等即可

+ 当地球同步轨道卫星的轨道倾角为0°时，即为地球静止卫星。地球静止轨道卫星在任何时刻都处于地面上同一地点的上方，地面观察者看到卫星始终位于某一位置，静止不动。
+ 当地球同步轨道卫星的轨道倾角大于0°并且小于 180°，但不等于90° 时，即为倾斜轨道同步卫星。其星下点轨迹是8字形。
+ 当地球同步轨道卫星的轨道倾角为90° 时，即为极地轨道同步卫星。由于轨道平面能够固定在惯性空间中，而地球在极轨道下方旋转，极地轨道同步卫星能够在低的轨道高度观测地面上的几乎每个点，所以常用于地球测绘或监视。

# 电场与磁场

+ 粒子通过一段圆弧，粒子在圆弧端点对应半径的直线上，在端点的中垂线上

### 回旋加速器
+ 交变电压周期T=2πm/qB
+ 相邻轨道半径(速度)之比sqrt(2k-1/2k+1),k为进入某D型盒的次数，故k越大比例越小，轨道半径增量越小
+ 带电粒子离开加速器的速度v=qBR/m，与加速电压无关
+ 粒子运动圈速数n=E/2Uq=qB^2R^2/4mU，运动时间t=nT=πBR^2/2U，与粒子自身无关
+ 由于随着带电粒子速度的增大，当速度接近光速时，据爱因斯坦狭义相对论可知，粒子质量增大，回转周期变大，而与交变电压周期不一致，使加速器无法正常工作，所以回旋加速器不能无限地对带电粒子加速。

# 机械波、简谐振动

+ 在一个周期内,物体运动的路程是4A,半个周期内,物体的路程是2A,但在四分之一个周期内运动的路程不一定是A

## 电学实验
+ 电表读数一般在1/3至2/3之间
+ 分压电路一般选用电阻小而额定电流大的电阻
+ 当限流电路无法使最小电流小于额定电流时，或待测电阻远大于滑动变阻器电阻，或需要电压范围变化大时，使用分压式接法
+ 电压和电流要求从零开始时，使用分压式接法，滑动变阻器选用与待测电阻大致相等的电阻
+ 分压与限流都可以用时，优先使用限流式以避免大电流放电
+ 根据sqrt(RaRv)来判断内外接
+ 误差分析需要写出表达式，然后判断测量值的偏大/偏小
## 测定玻璃砖折射率
+ 插针法：仅下表面不对齐时n偏大，仅上表面不对齐时n偏小，上下平移时n无误差
+ 视深法：n=实深/视深

# 物理大题规范表达
## 总纲
+ 写原始公式并标号
+ 结果先写表达式再代入数据计算”代入数值得“，可以检查量纲
+ 注意符号准确性（尤其是大小写）
+ 写出关键结论/临界条件
+ 动量和速度标定方向
+ 设未知量或标在图上（“各量如图中所示”），除明显物理量外（比如Va）
+ 多解要全部写出并讨论是否舍去
+ 说明研究对象：“对A：”，“对A，B：”
+ 说明研究过程：“A->B：”
+ 说明使用定理：“由动能定理/牛顿第二定律得”
+ 解答“解①②③式得”，“将xxx代入”
+ 因为所以用文字表示
+ 坐标系的方向用“与x轴正方向夹角为xxx”表示
+ 几何关系直接"由几何关系得"
+ 求受力：注意是否需要用牛顿第三定律
+ 用机械能/能量守恒：注意标定零势能面
+ 用动量：标定正方向
