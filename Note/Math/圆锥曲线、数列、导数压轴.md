# 圆锥曲线
## 化简技巧
+ 根据直线斜率是否存在和过定点设方程
+ 化简时带入直线方程而不是同时算x1+x2和y1+y2
+ 对目标式尽可能化简后再计算
+ 面积的表达式尽可能用割补或铅锤高计算
+ 注意相似的运用（注意面积比要平方！！）
+ 已知两根之比x1/x2：构造x1/x2+x2/x1+2=(x1+x2)^2/x1x2
+ 构造t=sqrt(x1)+sqrt(x2)：t^2=x1+x2+2sqrt(x1x2)
+ 一般使用关于y的弦长公式来表达焦半径（因为F的y为0）
+ 尽可能代换出用韦达定理表示的形式
+ 与圆上点有关的问题一般使用垂直解决
+ 注意题目的几何性质
+ 尽可能直接解出y1,y2(常数)
## 点差法
+ 一般用于弦中点，可以得到直线斜率和中点与O斜率的关系
## 点乘双根法
+ 用于需要算出对称式(mx1+n)(mx2+n)的值(一般是点乘证垂直)，它可以用韦达定理算出，但点乘双根法更快
+ 将联立后的方程ax^2+bx+c=0表示为a(x-x1)(x-x2)=0=ax^2+bx+c，然后代入特定的x0（一般是已知定点坐标），可以直接得出(x0-x1)(x0-x2)的表达式
+ 不需要同时算出(x0-x1)(x0-x2)和(y0-y1)(y0-y2)，若y=kx+b，则可以令x=(y0-b)/k代入后化简得到(y0-y1)(y0-y2)，注意提前讨论k=0的情况
## 非对称韦达
+ 对于y1y2不对称的式子，先考虑将y1y2代换为y1+y2，然后可能会出现常数比例
+ 否则将y1用y2和y1+y2表示并把y1y2代入化简
## 抛物线
+ 抛物线上两点表示的斜率比较容易化简(用y^2/2p代换x，然后平方差公式消掉y1-y2)
+ 通常用抛物线方程而不是直线方程代入消元
+ 抛物线设点可以用参数方程
+ 由于抛物线上两点斜率容易知道，可以直接写y-y1=k(x-y1^2/2p)后消元
+ 用求导求抛物线的切线方程
## 连线过定点问题
+ 一般设直线方程，解出直线方程的系数关系，得到定点，注意要考虑特殊情况
## 极坐标方程
+ 一般用来计算与焦点弦长度、面积相关问题
+ 以某个焦点为极点，使用第二定义推导极坐标方程
+ 焦点弦的两段相加后化简得到一条焦点弦的长度，另一条直接通过角的代换得到
## 齐次化
+ 目标是将k1,k2视作ak^2+bk+c=0的两根，用于已知k1+k2或k1k2的情况
+ 如果直线过的定点不在原点，通过平移变换把定点作为原点，**注意最后的答案要变换回来**
+ 设直线方程为mx+ny=1，与相交的曲线联立，得到只有二次项和一次项的方程，用1的代换将一次项升次，然后同时除以x^2得到关于k的方程
+ 利用已知条件求出m、n的关系，进而得到直线方程过定点
## 最值/取值范围问题
+ 要注意Δ限制了参数的取值范围
+ 一般化为均值不等式的形式
+ 很大串的式子考虑令分子为t然后分母变为均值不等式的形式
+ 尝试先对目标函数化简换元再代入韦达式
# 数列
## 倒数裂项
+ 对于a_{n+1}=a_n(ca_n+1)的形式，可以取导数裂项为1/(a_n+1/c)=1/a_n-1/(a_n+1)
## 积分定界
+ 对于恒凹/凸的函数，使用积分来得到上下界
## 几何级数
+ 如果数列递减且小于一个无穷递减几何级数，则使用a1/(1-x)来放缩
## 蛛网图
+ 对于a_{n+1}=f(a_n)的形式，可以视为FPI问题，通过y=x与y=f(x)的迭代图像观察数列的收敛性与a1的关系，同时利用求导得到不动点的稳定性
+ 对于g(a_{n+1})=f(a_n)的形式，同时画出y=g(x)与y=f(x)的图像迭代即可
## 局部不等式
+ f(n)< sigma(a_i)，可以通过差分证明f(n)-f(n-1)< a_n
+ f(n)< prod(a_i)，可以通过两边取对数再差分或证明f(n)/f(n-1)< a_n
## 伯努利不等式
+ 对于x>-1,n>=1，(1+x)^n>1+nx
+ 对于x_i>-1,n>=1，prod(1+x_i)>1+sigma(x_i)
## misc
+ 碰到a_n+1/a_n的形式可以考虑两边平方得到不等式
+ 如果上下界不够紧则分出前几项正常累加
+ 如果上下界与n有关则可以使用局部不等式或数学归纳法证明
+ 如果数列通项仅与n有关则放缩为和n、n+1有关的式子
# 导数
## 隐零点
+ 一般用于导函数零点不易找到的情况
+ 首先根据导函数的特点找到零点的取值范围（证单调+零点存在性定理）
+ 然后对导函数变形（两边取指数或对数），套入求极值
+ 根据化简后的结果（可能凑成均值不等式）得到范围
+ 也可以利用极值点本身的地位（极大值/极小值）与f(c)比较来证明边界
+ 含有参数的，把参数换成由x0表示的形式代入求极值
## 清君侧
+ 尽可能保持lnx独立，以减少难度和求导次数
+ 尽可能保持e^x与其它函数杂在一块，以减少求导次数
## 三角函数
+ 敢于多次求导
+ 利用三角函数的有界性
+ 取三角函数的特殊点
## 切线放缩
+ e^x和lnx同时存在时可以考虑放缩（尤其是对lnx放缩）
## 凹凸构造
+ 构造函数为f(x)>=g(x)的形式，然后证明f(x)min>=g(x)max
+ 一般将钩形函数和单调增函数通过减去一个kx来把单调函数掰弯，k的值由不易求极值点的一边构造
## 凹凸性反转
将f(x)拆为f1(x)+f2(x)+...+fn(x)，然后f(x)>=f1(x)min+f2(x)min+...+fn(x)min
## 极值点偏移
+ 一般等值点x1,x2在极值点x0两侧，证明x1+x2>2x0
+ 不妨设x1< x0 < x2，构造函数H(x)=f(x0+x)-f(x0-x)（对称比较好做），证明当x>0时H(x)<0即f(x0+x)< f(x0-x)
+ 然后把x1的对称换到x0右边用上面的结论+x0右边单调证明
## 同构证单调
+ 尝试构造同构（尤其是以e的指数部分为基准，有时会把系数丢上去）
+ 证同构的函数单调，故两个表达相等，进一步得到关于x0的更多信息
## 换元
+ 构造出x1/x2的形式，然后令其为t构造新函数(对于x1+x2，x1-x2之类的，如果x1+x2=c或x1x2=c都可以解出x2=h(t)再代入)
+ 只有lnx和x的可以考虑换元，参变分离比较好做
## 参数的分类界点
+ 代入特殊点（端点、导数零点）进行必要性探路
+ 参变分离+洛必达得到范围后再分类
## misc
+ 注意凑出第一小题的形式
+ 尽可能凑出基本不等式或二次函数的形式
+ 考虑换主元
+ 考虑两边同时取对数或乘e^x来减少单调性的讨论
