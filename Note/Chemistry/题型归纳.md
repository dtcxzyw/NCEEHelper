# 总纲
+ 无法定性判断时一定要写方程式
+ 注意物料守恒、氧化还原得失电子守恒、质子守恒、电荷守恒、质量守恒
+ 注意说明原因时要看问题是否有多个，重点是什么
+ 一定要尽可能用到题目给的所有带化学术语的条件（可用打勾）
+ 判断题可用错误的做法得到与选项相同的答案，然后将其排除
+ 注意多重反应间的能力优先级（氧化还原、酸碱中和、沉淀）,但浓度的影响因素更大
# 基本概念
+ 单质:由同种元素组成的纯净物,元素以单质形式存在时的状态称为元素的游离态。
+ 同素异形体:同素异形体是指由同样的单一化学元素组成，因排列方式不同，而具有不同性质的单质。
  碳的同素异形体：金刚石、石墨、富勒烯、碳纳米管、石墨烯和石墨炔；磷的同素异形体：白磷和红磷；氧的同素异形体：氧气、臭氧、四聚氧和红氧。硫的同素异形体：斜方硫、单斜硫（熔沸点不高）
+ 同分异构体:化学上，同分异构体是一种有相同分子式而有不同的原子排列的化合物。
+ 氧化物:组成中只含两种元素，其中一种一定为氧元素的化合物。过氧化物属于氧化物。
+ 电解质：电解质是溶于水溶液中或在熔融状态下就能够导电的化合物。CaCO3是强电解质
+ 同位素：具有相同质子数，不同中子数的同一元素的不同核素互为同位素。（16O2与18O2的混合物是由不同原子组成的单质）
+ 同系物:指结构相似、分子组成相差若干个“CH2”原子团的有机化合物；一般出现在有机化学中，且必须是同一类物质（含有相同且数量相等的官能团，羟基例外，酚和醇不能成为同系物，如苯酚和苯甲醇）。注意官能团与其他原子的连接方式也要相同。
+ 酸性氧化物是一类能与水作用生成酸或与碱作用生成盐和水的氧化物。（不能只有前半句，如SiO2）
+ 纯净物：由一种单质或一种化合物组成的物质（高分子聚合物如聚乙烯是混合物而不是纯净物，因为相对分子质量不定）
# 化学与生活

+ 石油的分馏是物理变化，液化、裂解是化学变化，煤的干馏、气化、液化都是化学变化
+ 生理盐水0.9%NaCl，医用酒精体积分数75%
+ 食品干燥剂：吸附（硅胶、氧化铝），化合（CaCl2,CaSO4）。生石灰干燥剂已被淘汰(因此也不行)。五氧化二磷、NaOH不能做干燥剂。
+ 草木灰和铵态氮肥不能混合使用
+ 二氧化硫可做葡萄酒的食品添加剂
+ 加碘食盐中的碘是KIO3（不是KI，因为KIO3较稳定）
+ 钡餐用BaSO4而不是BaCO3
+ 浓硫酸沾在衣服或皮肤上，立即用大量的水冲洗，然后再涂上碳酸氢钠溶液（用布擦会扩大擦伤面）
+ 高温灭病毒与蛋白质有关
+ 灼烧法可区分蚕丝和人造纤维
+ 甘油具有吸水性，可做护肤保湿剂
+ 食用油反复加热会产生有害物质
+ 聚乙烯可用于食品包装而聚氯乙烯不能
+ 阿司匹林水解生成酸，用NaHCO3解毒
+ 制造肥皂：油脂在碱性条件下水解为高级脂肪酸盐和甘油
+ 金属着火时不能用泡沫灭火器灭火
+ 焰色反应可用于制造烟花
+ 碳在常温下具有稳定性，不易反应，故古代名画现代能保存，书写档案要用碳素墨水
+ 人体含氟主要分布在骨骼、牙齿中
+ O3可杀菌、消毒、漂白
+ 地壳中的元素含量：O、Si、Al、Fe
+ 注意盐酸是混合物
+ SO2是化合漂白，HClO是氧化漂白
+ 青花瓷、分子筛、玉的成分是硅酸盐
+ N2H4和N2O4可作为火箭推进剂的原因是反应放出大量的热、产物不污染环境、放出大量气体
+ 古代炼银为AgO加热分解，炼汞为HgS加热分解
+ “近朱者赤”指HgS
+ 钛形成氧化膜具有抗腐蚀能力，熔点高，密度小，被称作继铁、铝后的第三金属，在自然界中储量大，但因生产成本高而仅用于尖端领域
+ 飞机播撒碘化银可人工降雨（作为凝结核）
+ 焊接钢材时，分别用饱和Na2CO3（除油脂）和NH4Cl（除锈）处理
+ 乙二醇可作为汽车防冻剂
+ 食盐、蔗糖溶解溶液温度一般不变，硝酸盐、铵盐溶解吸热，其余一般放热
+ 侯氏制碱法利用NaHCO3溶解度较小的原理
+ 注意单糖不能水解
+ 氮的固定主要通过生物固氮（根瘤菌）
+ 硫黄可做酸性肥料、加工天然橡胶、医治皮肤病
+ 常量元素：氯钠镁硫钙钾碳锶溴硼氟，溴被称为海洋元素
+ 新型陶瓷：氮化硅Si3N4、碳化硅SiC、生物陶瓷(如ZrO2)、压电陶瓷
+ 黑色金属：铁铬锰及其合金，其他金属称为有色金属
+ 生铁->钢减少了C并增加Si、Mn等因素，不锈钢在钢的基础上加入铬、镍（铬：形成氧化膜，镍：化学稳定性和抗腐蚀性）
+ 金（黄色）银（银白色）铜（紫红色）都具有延展性、导电性、导热性（延展性金最强，导电导热性银最强）
+ 王水（浓盐酸、浓硝酸以体积之比为3：1混合，浓度不确定，组成不确定）
+ 焰色反应不是化学反应
## 天然矿石
+ 自然界存在的刚玉即属于α一Al₂O₃，它的硬度仅次于金刚石
+ 大理石、方解石、白云石、菱锌矿、菱镁矿、菱铁矿都是碳酸盐矿石
+ Al₂O₃由于含有不同的杂质而有多种颜色。例如含微量Cr（III）的呈红色，称为红宝石；含有Fe（II），Fe（III）或Ti（IV）的称为蓝宝石。
+ 明矾KAl(SO4)2·12H2O，黏土（Al2O3·2SiO2·2H2O）制成陶器
+ 石膏（CaSO4·2H2O）、芒硝（Na2SO4·10H2O）、重晶石（BaSO4）、矾石[(AlO)2SO4·9H2O]、明矾石[K2SO4·Al2(SO4)3·24H2O]
+ 钾盐以硝石（硝酸钾，KNO3），明矾（十二水合硫酸铝钾，KAl(SO4)2·12H2O），还有草木灰（碳酸钾，K2CO3）的形式
+ 雄黄As4S4，雌黄As2S3，三氧化二砷被称为砒霜，用银检测
+ 天然的硫化汞又称为朱砂，由于具有鲜红的色泽，因而很早就被人们用作红色颜料。汞泄漏时撒硫磺粉，硫磺粉和水银在常温下可以反应生成“硫化汞”。
## 热塑性与热固性
+ 热塑性是指物质在加热时能发生流动变形，冷却后可以保持一定形状的性质。大多数线型聚合物均表现出热塑性，在一定温度范围内，能反复加热软化和冷却硬化。它们可以通过加热熔化来进行封口、粘合等操作。代表物有聚乙烯（PE ）、聚氯乙烯（PVC）、聚丙烯（PP ）
+ 热固性指加热时不能软化和反复塑制，也不在溶剂中溶解的性能，体型聚合物具有这种性能。代表物有酚醛树脂（电木）。
## 硬水与软水
+ 软水是指不含或含较少可溶性钙、镁化合物的水。自然水中，雨水和雪水属软水，常见的有蒸馏水。硬水是指含有较多可溶性钙、镁化合物的水。
+ 鉴别：倒入肥皂水，轻轻搅拌。水面上出现泡沫的为软水，水面上出现浮渣的为硬水，浮渣越多，水的硬度越大。硬脂酸钠变为硬脂酸镁和硬脂酸钙沉淀。
+ 鉴别：用烧杯加热，在杯壁留下较多水垢的是硬水。加热后，可溶性的钙镁物质转化成不可溶性的物质。
+ 软化：煮沸、离子交换、化学软化法
## 胶体
+ 溶质叫分散质，溶剂叫分散剂
+ 直径在1~100nm间，胶体透明且均匀稳定
+ 气溶胶：烟、雾，液溶胶：蛋白质、淀粉、Fe(OH)3，固溶胶：水晶、有色玻璃
+ 蛋白质、淀粉、淘米水是分子胶体，Fe(OH)3、碘化银是粒子胶体
+ 具有丁达尔现象，聚沉（加入可溶性盐，离子与胶体粒子电荷中和形成沉淀，如氢氧化铁胶体聚沉，豆腐制作）、电泳（吸附同电性粒子，形成带电微粒，在电场下定向移动）、渗析（半透膜与滤纸区别）等性质
+ 泥土悬浊液、乳浊液不是胶体
## 环境保护
+ 光化学烟雾：碳氢化合物和氮氧化合物在紫外线下生成有毒物质
+ 温室效应：与CO2、CH4、H2O、N2O、氟利昂等有关
+ 酸雨：pH<5.6，与硫氧化物（SO2）和氮氧化物有关，CO2排放不是酸雨的主因，减少酸雨形成的方法有：控制酸性物质排放、改变能源结构开发清洁能源
+ 水体富营养化、赤潮、水华：N、P含量过多导致藻类等过度繁殖
+ 臭氧空洞:氟利昂、氮氧化物催化臭氧为氧气
+ 空气污染：可吸入颗粒物、氮氧化物、SO2
+ PM2.5:<2.5微米，不是胶体
## 复合材料
+ 基体：粘结作用，增强体：骨架作用
+ 玻璃钢：玻璃纤维做增强体，合成树脂做基体
+ 碳纤维增强复合材料：碳纤维做增强体，合成树脂做基体
# 阿伏伽德罗常数
+ 注意气体是否为标况（不为标况也可能正确）
+ 注意是否给定溶液体积
+ 标况下不为气体的：HF,NO2,H2O,SO3,Br2,N2O4,大多数有机物，乙醇，硫酸，CCl4，苯，甲醇
+ 标况下为气体的有机物：HCHO,C<=4的烷烃或一氯代烃
+ 化学键数：1molP4中P-P键有6mol,1molS8中有8molS-S键，1molC(金刚石)有2mol C-C键（正四面体），1mol C(石墨)有1.5molC-C键（六边形）
+ Na2O2、CaC2中的是O22-离子，C22-离子
+ 注意浓变稀的问题，因此浓酸不可能反应完
+ 注意电解精炼铜时阳极是粗铜，还有其它金属变为离子
+ 浓酸浓度：浓硫酸18.4mol/L 浓硝酸16mol/L 浓盐酸12mol/L
+ 注意计入溶剂的问题（有时也可能正确）
## 与浓硝酸的反应
+ 冷的浓硝酸能使铁铝钝化（形成致密氧化膜），热浓硝酸的不会使其钝化。
+ 能与铜反应（不加热），不生成氢气
+ 与非金属CPS反应要加热
## 与浓硫酸的反应
+ 常温下，浓硫酸能使铁、铝等金属钝化（形成致密氧化膜）。
+ 常压加热时，浓硫酸可以大多数金属反应，生成高价金属硫酸盐，本身被还原成SO2，S，H2S或金属硫化物，浓硫酸表现酸性和氧化性
+ 热的浓硫酸可将CPS等非金属单质氧化到其高价态的氧化物或含氧酸，本身被还原为SO2。浓硫酸表现氧化性
## 常见可逆反应
+ 弱电解质水解、电离、胶体生成
+ H₂(g)+I₂(g)⇌2HI(g)
+ Cl₂+H₂O⇌HCl+HClO
+ N₂+3H₂⇌2NH₃
+ 2SO₂+O₂⇌2SO₃
+ 2NO2⇌N2O4
## 特殊氧化还原反应
+ Fe、Cu与S，I2反应仅到Fe2+,Cu+(加热)
+ Fe与Cl2、HNO3反应到Fe3+
+ Na2O2、H2O2分解自身氧化还原
# 有机化学基础
+ 注意氯气取代甲烷时有副产物HCl
+ 几乎任何有机物都可以发生氧化反应（燃烧），但CCl4不易燃用作灭火剂
+ 还原/加成反应消耗H2=不饱和度（不容易开环，故环不算）
+ 与NaHCO3反应只有-COOH，与Na2CO3反应有-COOH，酚羟基，与NaOH反应有-COOH，酚羟基，-COO-,与Na反应有-COOH，-OH(~0.5H2)
+ 1-COO-酸性条件下水解消耗1H2O
+ 注意是否可能成环，C不够时N、O也可在环上
+ 注意考虑醚键、酮羰基、-OOCH（与银氨溶液反应，可水解，C、O不够）、水解后的酚羟基（水解后还能消耗NaOH）
+ 氨基酸是含有碱性氨基和酸性羧基的有机化合物，α-氨基酸为天然氨基酸，其氨基与羧基在同一个碳上
+ 植物油是由不饱和脂肪酸和甘油化合而成的化合物，故可使酸性高锰酸钾溶液褪色；动物油主要含有饱和脂肪酸,故熔点很高
+ 注意石油分馏产物是混合物
## 官能团/有机物的检验
+ 碳碳双键/叁键：使溴的CCl4溶液（加成）和酸性高锰酸钾溶液（氧化还原）褪色
+ 羟基：与Na反应有H2放出
+ 羧基：与NaHCO3反应放出CO2，溶解新制氢氧化铜的絮状沉淀
+ 苯酚：与FeCl3发生紫色反应
+ 淀粉：遇碘变蓝
+ 葡萄糖（醛基）：与银氨溶液发生银镜反应（R-CHO + 2[Ag(NH3)]2OH =加热= R-COONH4 + 3NH3 + 2Ag(沉淀) + H2O），与新制氢氧化铜溶液反应生成砖红色沉淀（CH3CHO+2Cu(OH)2-水浴热→CH3COOH+Cu2O↓+2H2O）（注意果糖、甲酸、甲酸酯也能发生银镜反应）
+ 蛋白质：与浓硝酸微热发生颜色反应（黄），灼烧有烧焦羽毛气味
## 有机物的互溶与分层问题
+ 相对密度：乙醇<甲苯<苯<水<硝基苯<四氯化碳<溴
+ 溴水的密度应该介于溴单质与水之间
+ 乙醇易溶于水，溴微溶于水
+ 溴溶于酒精、四氯化碳、苯、甲苯：红棕色
+ 苯，甲苯，四氯化碳，硝基苯都可以从溴水中萃取溴单质
+ 水和除乙醇之外的有机溶剂会分层。
+ 相似相溶原理：含有羟基的可以相溶，但是如果碳链很长羟基很少的除外：氢键的作用使得水、乙醇、甲醇、甘油等能够以任意比例互溶。
+ 分液中苯在上层，水在下层；溴苯在下层，水在上层；四氯化碳在下层，水在上层
# 实验
+ HClO不能用pH试纸
+ pH试纸只能精确到整数
+ Mn的离子或化合物起催化作用
+ 注意分液漏斗从下口出用“放”，从上口出用“倒”
+ 喷泉实验：水+NH3、HCl、NO2,NaOH+CO2、Cl2、SO2，酸性高锰酸钾+SO2，NH3+HCl,O2+白磷
+ 电石(CaC2)加水制备乙炔有杂质H2S等
+ 洗气时浓硫酸要放在最后
+ 铝热反应时蒸发皿上铺细沙的作用是防止蒸发皿炸裂和防止熔融液体溅出
+ 制备Fe(OH)2时要煮沸NaOH溶液和铺油膜防氧化，胶头滴管应插入试管底部
+ 制备氨气时不能使用NH4Cl和NH4HCO3加热制备
+ SO2使酸性高锰酸钾、溴水、碘水褪色体现还原性，使NaOH的酚酞溶液体现自身为酸性氧化物，其余才体现漂白性
+ 中和热实验中为了保证酸碱完全中和碱要稍过量（因为NaOH容易吸收CO2），溶解会放热，弱酸弱碱电离会吸热，中和热-57.3kJ/mol
+ 乙醇催化氧化实验中铜丝下端绕成螺旋状可以增大接触面积，加快反应速率，铜丝被氧化为CuO，CuO再与乙醇反应生成Cu和乙醛
+ 乙酸乙酯的制备中浓硫酸不能最先加入，浓硫酸起催化剂和吸水剂的作用，饱和Na2CO3起吸收乙酸、溶解乙醇、降低乙酸乙酯溶解度便于分层的作用
+ 溴苯的制备中应先加苯后加液溴（减少溴挥发），除去溴苯中的溴可加入NaOH溶液，振荡，再用分液漏斗分离，起催化作用的是FeCl3。
+ 制备胶体时不能搅拌
+ 富集仅指提高所需物质浓度，但不必考虑杂质
+ 酸性重铬酸钾用浓硫酸而不是浓盐酸配置（浓盐酸会被氧化）
## 干燥试剂的选用
+ 酸性干燥剂：浓硫酸，P2O5(粉末)，硅胶（多孔固体，含钴时吸水蓝色变粉红色）
+ 碱性干燥剂：碱石灰（CaO+NaOH）,CaO,NaOH
+ 中性干燥剂：无水CaCl2，Al2O3,无水CuSO4（仅用于检测水，吸水后变蓝）
+ 碱性干燥剂不能干燥酸性气体、酸性干燥剂不能干燥碱性气体（NH3）
+ 浓硫酸不能干燥还原性气体(H2S,HI,HBr)，乙烯（常温下反应）,但可以干燥H2,CO(它们的还原性在高温下体现)，SO2(相邻价态不发生氧化还原反应)
+ 无水CuSO4不能干燥H2S(产生CuS沉淀)，不能干燥NH3(形成络合物)
+ 无水CaCl2不能干燥NH3(形成加合物)
## 试剂的保存
+ 不能磨口玻璃瓶保存NaOH（生成粘性硅酸钠）
+ 氢氟酸不能保存于玻璃瓶中（生成SiF4）
+ 浓硝酸保存于棕色试剂瓶中（久置浓硝酸生成NO2溶解显黄色）
+ 保存液溴时为防止溴的挥发加水液封
+ 钠保存在煤油中（与水反应，防氧化，密度高于煤油，多余的钠可以放回原瓶），锂保存在固体石蜡中（密度低于煤油）
+ 冷浓硫酸、冷浓硝酸用铁质或铝质容器储存

## 滴定指示剂的选用
+ 推断滴定终点时，先考虑滴定完全，再多加一点标准液，考虑溶液变化
### 酸碱中和滴定
+ 指示剂变色要明显.溶液颜色的变化由浅到深容易观察，而由深变浅则不易观察。
+ 滴定终点时与指示剂变色范围相一致.
+ 强酸滴弱碱：终点为酸性，甲基橙
  强碱滴弱酸：终点为碱性，酚酞
  强酸滴强碱：酚酞或甲基橙（酸滴碱时甲基橙，碱滴酸时酚酞）
+ 石蕊溶液由于变色范围较宽，且在滴定终点时颜色的变化不易观察，所以在中和滴定中不采用。
### 氧化还原滴定
+ 碘量法用淀粉，酸性高锰酸钾自身指示
+ 注意比较反应完前和反应完后的溶液内粒子形式，确定终点的颜色变化
### 沉淀滴定
+ 一般使用银量法测定卤素离子的含量
+ 到达滴定终点时，Ag+会与指示剂反应生成有色沉淀，以此指示终点

## 金属性和非金属性
### 金属性
+ 与水或酸反应置换氢的难易
+ **最高价**氧化物对应水化物的碱性强弱
+ 单质的还原性或离子的氧化性
+ 单质间的置换反应
+ 电化学电极判断
+ 元素在周期表中的位置
### 非金属性
+ 一般情况下，氧化性越强，对应非金属性越强。
+ 由单质和水生成酸的反应程度判断：反应越剧烈，非金属性越强。
+ 由对应氢化物的稳定性判断：氢化物越稳定，非金属性越强。（反例：甲烷比氨稳定）
+ 由和氢气化合的难易程度判断：化合越容易，非金属性越强。
+ 由**最高价**氧化物对应水化物的酸性来判断：酸性越强，非金属性越强。（反例：硒酸的酸性强于硫酸）
+ 由对应阴离子的还原性判断：还原性越强，对应非金属性越弱。
+ 由置换反应判断：强置弱。
+ 元素在周期表中的位置
## 物质的溶解度
+ 溶解度明显受温度的影响，大多数固体物质的溶解度随温度的升高而增大；气体物质的溶解度则与此相反，随温度的升高而降低。
+ 固体和液体的溶解度基本不受压力的影响，而气体在液体中的溶解度与气体的分压成正比。
+ 多数固体物质的溶解度随温度的上升而增大，如氯化铵、硝酸钾。
+ 少数物质的溶解度受温度变化的影响很小，如氯化钠。
+ 含有结晶水的硫酸钠（Na₂SO₄·10H₂O）的溶解度开始随温度的升高而增大，当达到一定温度（32.4℃）时，随温度的升高而减小（这时Na2SO4·10H2O脱水成Na2SO4）。
+ 含有结晶水的氢氧化钙[Ca(OH)2·2H2O]和醋酸钙[Ca(CH3COO)2·2H2O]等物质的溶解度随温度的升高而减小。
+ 硫酸锂的溶解度随温度的升高而减小。
+ 气态氢化物：CH4(极难溶)，NH3(极易溶)，H2S(可溶)，HCl(极易溶)
+ 相对分子质量越大，有机酸越难溶于水。
+ 硫,白磷可溶于CS2
### 记忆
+ K+、Na+、NH4+、NO3-一般都可溶
+ OH-：Ca2+（微溶）、Ba2+（可溶）
+ Cl-：Pb2+（微溶）、（Hg+、Ag+）（不溶）
+ SO42-：（Ca2+、Ag+、Hg+）（微溶）、（Ba2+、Pb2+）（不溶）
+ S2-：除K+、Na+、NH4+、H+外均不存在或不溶
+ SO32-：Mg2+(微溶)、除K+、Na+、NH4+、H+外均不存在或分解或不溶
+ CO32-：Mg2+(微溶)、除K+、Na+、NH4+、H+外均不存在或分解或不溶
+ SiO32-：除K+、Na+外均不存在或分解或不溶(H+也不溶)
+ F-:常见MgF2、CaF2、BaF2、PbF2、AlF3、CuF2
+ 碳酸氢钙微溶
## 物质的熔沸点
+ MgO、Al2O3熔点高，MgO>Al2O3
+ 离子化合物比共价化合物熔点高（AlCl3为共价化合物,熔融时不导电，不能被电解，但MgCl2可以）
+ 组成和结构不相似的物质（相对分子质量相近），分子极性越大，其熔沸点就越高，如熔沸点CO＞N2，CH3OH＞CH3CH3。
+ 组成和结构相似的分子晶体，相对分子质量越大，分子间作用力越强，物质的熔沸点越高。如：CH4＜SiH4＜GeH4＜SnH4。
+ NH3，H2O，HF有氢键，熔沸点反常升高
### 晶体
+ 原子晶体原子间键长越短、键能越大，共价键越稳定，物质熔沸点越高，反之越低。如：金刚石（C—C）＞碳化硅（Si—C）＞晶体硅 （Si—Si)
+ 离子晶体中阴、阳离子半径越小，电荷数越高，则离子键越强，熔沸点越高，反之越低。如KF＞KCl＞KBr＞KI，CaO＞KCl。
+ 分子晶体分子间作用力越大物质的熔沸点越高，反之越低。（具有氢键的分子晶体，熔沸点反常地高）如：H2O＞H2Te＞H2Se＞H2S，C2H5OH＞CH3OCH3。
+ 原子晶体＞离子晶体(Al2O3)＞分子晶体(AlCl3)
+ 金属晶体原子半径越小，金属阳离子与自由电子静电作用越强，金属键越强，熔沸点越高，反之越低。如：Na＜Mg＜Al。
### 有机
+ 高级脂肪酸形成的油脂中，不饱和程度越大，熔沸点越低。
+ 烃、卤代烃、醇、醛、羧酸等有机物一般随着分子里碳原子数增加，熔沸点升高，如C2H6＞CH4，C2H5Cl＞CH3Cl，CH3COOH＞HCOOH。
+ 同分异构体：链烃及其衍生物的同分异构体随着支链增多，熔沸点降低。
+ 芳香烃的异构体有两个取代基时，熔点按对、邻、 间位降低。沸点按邻、间、对位降低。
# 化学电源与电解池
+ 注意溶剂是否是水（固态氧化物的放电离子是O2-）
+ 注意电子、电流、离子的流向
+ 注意离子交换膜
+ 注意电荷守恒导致的离子移动，从而导致质量变化与电极质量变化不同
+ 注意非电化学反应的发生
+ 注意是原电池还是电解池，注意充电还是放电
+ 一般O2处为正极
+ 注意电解质溶液的酸碱性，确定离子存在形式
+ 微生物原电池：注意环境pH和温度改变使生物失活的情况
## 电化学的应用
+ 电渗析：离子在电场作用下穿过离子交换膜
+ 电镀：镀层金属做阳极（阳极也可以是惰性电极但电解液金属离子无法得到补充）并存在于电解液中，待镀工件做阴极
### 防腐蚀
+ 加入缓蚀剂或覆盖层保护
+ 牺牲阳极的阴极保护法：用还原性强的物质做阳极，被保护物质做阴极
+ 外加电流的阴极保护法：被保护物质做阴极，另一导体做阳极（仅起导电作用但不损耗）
### 电解精炼铜
+ 阳极为粗铜，阴极为纯铜，电解液一般为硫酸铜溶液
+ 阳极会有其它金属先于铜放电，并产生阳极泥，因此不能根据流过电子数推断阳极减重
### 氯碱工业
+ 阳极生成Cl2，阴极生成H2,NaCl溶液变为NaOH(在阴极区)
+ 使用阳离子交换膜:使用阴离子交换膜会导致Cl-移动影响纯度，Cl2会与NaOH反应；不使用膜可能导致H2与Cl2混合爆炸
## 常见电池
+ 酸性锌锰干电池：（－）Zn|NH4Cl（20%）ZnCl2|MnO2，C（+）  正极2MnO2+ 2NH4++2eˉ→2MnO(OH) + 2NH3 负极 Zn+2NH3 - 2eˉ→Zn(NH3)2 2+
+ 碱性锌锰干电池：（－）Zn︱KOH，K2[Zn(OH)4]︱MnO2，C（+） 正极MnO2+H2O+eˉ→MnO(OH)+OHˉ  负极Zn＋2OH－-2e→baiZn(OH)2
+ 铅蓄电池：(-) Pb |H2SO4|PbO2(+) 放电时总反应归中生成PbSO4 充电时总反应歧化电极恢复
+ 氢氧燃料电池（要考虑介质）：H2通负极，O2通正极，电极为惰性电极但有催化活性，总反应相当于氢气与氧气燃烧生成水，但无明火
## 电化学中两极得失电子顺序
+ 阳极：活泼金属阳极（金属活动顺序表中银或银以前的金属）>S2-＞I -＞Br-＞Cl-＞OH－＞高价含氧酸根(水溶液中不放电) ＞F－

# 化学工艺流程
+ 一般选用绿色（不引入杂质）氧化剂H2O2(H2O2做氧化剂时生成H2O/OH-，做还原剂时生成O2)

+ 注意写的是电极反应式、化学方程式还是离子方程式

### 盐类蒸发产物
+ 难挥发性弱碱对应的离子：Al3+/Fe3+/Cu2+

## 预处理
+ 酸溶一般剩下SiO2，注意加硫酸生成CaSO4的情况
+ 煅烧：使固体分解，分离杂质（石灰石,硫铁矿）/改变结构（如高岭土煅烧后易电解）
+ 洗去表面油脂的方法为碱煮水洗
+ 研磨（固）、雾化（气）：增大反应物接触面积，提高反应速率和产率
+ 浸取：溶解固体、分离杂质
+ 打磨/化学表面处理：除去表面氧化物，提高光洁度，除去表面可溶性杂质
## 产物分离提纯
+ 过滤：固液分离
+ 结晶：溶解度不同，通过调节浓度和温度分离
+ 蒸馏：利用沸点不同
+ 萃取：利用物质在萃取剂中的溶解度不同
+ 分液：分离互不相溶的液体

+ 盐析：加入浓无机盐分离有机物（如蛋白质，但要注意不要使其变性）
+ 洗气：易溶气体与难溶气体分离
+ 重结晶：固体->固体提纯

+ 加氧化剂的目的：氧化xxx/使xxx完全氧化

### 特定温度条件的原因回答
+ 选择某温度的原因：温度过低xxx，温度过高xxx
+ 考虑副反应、平衡移动、溶解与结晶、催化剂活性
+ 升温会使气体逸出（降低气体溶解性），使某物质挥发或分解
+ 加热煮沸：促进水解，利于过滤分离
+ 降温：防止物质高温分解或挥发，减少能源成本，降低对设备要求
+ 高温下某些烃类物质会分解
### 洗涤的目的
+ 水洗：洗去晶体表面水溶性杂质
+ 冰水/有机试剂洗：洗去晶体表面杂质离子，且减少晶体的溶解损耗
+ 水洗后醇洗：洗去晶体表面杂质，减少晶体溶解，除去晶体表面水分
### 催化剂活性/产率异常下降的原因
+ 温度变化影响催化剂活性
+ 催化剂分解或表面覆盖物质（CaCO3，CaSO4）造成催化剂失活
+ 高温下产物分解
# 化学反应原理
+ 观察产品的氧化还原
+ 观察产品在滤液中还是滤渣中
+ 可根据活化能推断反应的ΔH
+ 可根据活化能大小和反应的速率表达式推断反应的决速步骤
+ 注意转化率是否为平衡转化率，如不是平衡转化率要考虑速率问题（如放热反应相同时间转化率随温度先升后降）
+ 解释现象时要写出可能存在的所有平衡，若有不利于解释的平衡，可以比较强度（比如末尾加上但电离程度大于水解程度）
+ 基元反应中k正、k逆和K一般有比例关系
+ 电离、水解均为吸热反应
+ 环状化学反应催化剂判断：找出反应物和生成物，生成生成物阶段的下一个环上物质即为催化剂。（催化剂作为第一个反应的反应物，最后一个反应的生成物，最后一个反应也生成总反应生成物）
+ 选择最佳条件时看第一个最高点/最低点
## 计算
+ 平衡常数的计算：尽可能直接用分压计算
+ 盖斯定律最好根据唯一出现的产物确定系数
+ 多步反应计算平衡常数时应设各物质的物质的量/分压，以及设每个反应的反应量，解方程后套公式
+ 根据初末质量/总压计算平衡常数时可使用整体差量法
+ 弱酸/弱碱盐能促进水的电离，酸式盐能抑制水的电离
+ 注意容器压缩/溶液稀释时体积变化为主
+ 计算时最好列出方程后通过观察整体代换计算，实在不行用上PV=nRT
+ 若反应与反应物浓度线性相关，浓度随时间变化一般为指数函数，故可用“半衰期”的概念计算
+ 中和热：在稀溶液中酸碱中和生成1mol水的反应热，燃烧热：1 mol纯物质完全燃烧生成稳定的化合物时所放出的热量
+ 注意水的气态/液态变化
+ ΔH=反应物键能-生成物键能，活化能低于反应物键能（不要全部断键）
+ 注意绝热条件，可视为先恒温反应后调整温度
+ 可以将水解/电离带来的原粒子浓度变化忽略（也可以用于不等式推导）
## 等效平衡（恒T）
+ 恒V，ΔV!=0：归边后完全相等，等效
+ 恒V，ΔV=0：归边后成比例，等效
+ 恒P：归边后成比例，等效
+ 充入两倍气体在恒容容器中反应可视为先扩容后加压
+ 溶液也可以类比思考（把体积看作液体体积）
## 条件变化引起平衡、速率变化
+ 温度变化：温度上升反应速率加快（要注意催化剂失活的情况），平衡向吸热方向移动
+ 压强变化：加压导致浓度上升，反应速率加快，平衡向体积减小的方向移动
+ 充入反应物：平衡正移，一般反应速率加快，多反应物的其它反应物平衡转化率上升，自身平衡转化率总下降
+ 移走生成物：平衡正移，一般反应速率减慢（除合成氨反应）
+ 充入惰性气体：看恒压还是恒容，恒容无影响
## 化学反应中间过程示意图
+ 能垒即活化能，是能量上升时的能量差
+ 最大最小能垒对应的化学方程式写前后的吸附态，不能写中间的过渡态Ts
+ ΔH看初末状态能量变化
## 原电池与电解池
+ 写反应方程式时注意不参与电化学过程的反应（比如氧化还原反应）
+ 注意质量变化中水的变化和离子的跨膜移动（比如H+）
+ 注意酸碱介质问题
+ 一般用总反应-阴极反应得到阳极反应方程式
+ 使用有机溶剂做电解液的优点是：便于Na+（H+/OH-后的离子）放电、导电性好、难挥发
### 电解类型及复原
+ 电解水：pH由溶液的酸碱性决定，溶液呈碱性则pH增大，溶液呈酸性则pH减小，溶液呈中性则pH不变。电解质溶液复原—加适量水。
+ 放氧生酸：pH变小。电解质 溶液复原—加阳离子相同的碱或氧化物。
+ 放氢生碱：pH变大。电解质溶液复原—加阴离子相同的酸。
+ 电解质：

  如果阴阳离子都可以参与反应,那么浓度降低 CuCl2

  如果阳离子可以得电子,阴离子不能失电子,得到酸 CuSO4

  如果阴离子可以失电子,阳离子不能得电子,得到碱 NaCl。

  电解质溶液复原—加适量电解质。

## 操作步骤
+ 洗涤沉淀：将沉淀置于过滤器中，加洗涤液至完全浸没沉淀，待洗涤液流尽后，重复2-3次
+ 检验沉淀是否洗涤干净：取最后一次洗涤液于试管，滴加xxx（检测NH4+时加热），不产生xxx现象(沉淀、气体、溶液变色)，说明沉淀已洗涤干净
+ 灼烧：将沉淀物继续加热,冷却至室温后称量,直至两次连续称量的质量差不超过0.1g
+ 检验装置气密性：微热法、液差法，注意要检查的是整套装置还是单个装置，注意关好止水夹或活塞
+ 滴定：左手控制滴定管活塞，右手振荡锥形瓶，眼睛注视锥形瓶内溶液颜色变化
+ 证明沉淀完全：静置，取沉淀后的上层清液，滴加xxx（沉淀剂），若没有沉淀生成则沉淀完全
+ 从溶液中得到晶体：蒸发浓缩、冷却结晶、过滤、洗涤、干燥
+ 蒸发结晶：将溶液转移到蒸发皿中加热并用玻璃棒不断搅拌，待有大量晶体出现时停止加热，利用余热蒸干剩余水分
+ 定容：向容量瓶加水至离刻度线1-2cm处，改用胶头滴管滴加，眼睛平视刻度线，滴加水至凹液面最低点与刻度线相切
+ 量气：待试管冷却至室温后，上下移动量气管有刻度一侧使量气管两侧液面相平，使视线与凹液面最低点相平读取数据
+ 测定pH：取pH试纸与表面皿（或洁净干燥玻璃片）上用洁净干燥玻璃棒蘸取待测液点在试纸中部，待其显色后与标准比色卡比色

# 基本元素性质
+ +4价的Pb有强氧化性
+ ClO-还原产物一般为Cl-
## Na
+ 工业制取：电解NaCl或NaOH
+ 钠与水反应：浮（密度比水小）溶（与水反应）游响（生成气体）红（反应生成碱）
+ Na2O不稳定容易被氧化为Na2O2
+ 钠和钾合金可做导热剂，钠可做高压钠灯，钠可用于钛钽铌锆的冶炼（高温下还原金属盐发生置换反应）
+ Na常温下氧化为Na2O，高温下Na2O氧化为Na2O2，在空气中最后变为Na2CO3
+ 工业制NaOH:电解饱和NaCl溶液/Na2CO3+Ca(OH)2
### NaHCO3和Na2CO3的比较
+ 加热判断稳定性（NaHCO3在内）
+ 比较同物质的量浓度时pH
+ 比较同物质的量浓度时与盐酸反应的速率
## Cl
+ Cl2能与Fe、Cu、H2（剧烈）、P(PCl3/PCl5)点燃反应，将金属氧化到最高价
+ 液氯储存于钢瓶中
+ 制取84消毒液（NaClO）：Cl2+NaOH==NaClO+NaCl+H2O，起作用时次氯酸盐与稀酸或空气中的CO2和H2O反应生成HClO
+ 制取漂白粉(Ca(ClO)2(有效成分)与CaCl2混合物)：2Ca(OH)2+2Cl2==Ca(ClO)2+CaCl2+2H2O
+ HClO见光易分解为HCl和O2
+ 高氯酸根ClO4-，氯酸根ClO3-，亚氯酸根ClO2-，次氯酸根ClO-
+ 实验室制取氯气：MnO2+4HCl(浓)=加热=MnCl2+Cl2↑+2H2O
+ 氯碱工业：2NaCl+2H2O=电解=2NaOH+H2↑+Cl2↑
+ ClO-始终体现强氧化性
+ ClO2无论等物质的量还是等质量的氧化能力均高于Cl2
## Al
+ 氢氧化铝不溶于氨水，AlCl3溶于氨水生成Al(OH)3
+ 铝热反应（铝的强还原性）：2Al+Fe2O3=高温=Al2O3+2Fe(Al和Fe2O3粉末的混合物称为铝热剂，放出大量的热，可用于焊接铁轨,铝热剂在最下面，上面铺层氯酸钾，中间插根镁条.反应时先点燃镁条，高温使氯酸钾分解产生氧气，这样又促进镁条燃烧.镁条燃烧产生大量热)
+ Al(OH)3能凝聚水中悬浮物和吸附色素
+ Al燃烧放出大量的热和耀眼白光，可做燃烧弹、信号弹、火箭推进剂
+ 制备Al(OH)3时用AlCl3和NH3·H2O反应，因为NH3·H2O不能使Al(OH)3变为[Al(OH)4]-
## Fe
+ Fe3+体现氧化性，Fe2+和Fe体现还原性
+ 配置FeCl2溶液时要加Fe防止氧化，配置FeCl3溶液时要先加HCl抑制水解
+ 3Fe+4H2O(g)=高温=Fe3O4+4H2
+ 制备Fe(OH)3胶体：FeCl3+3H2O=加热=Fe(OH)3(胶体)+3HCl
+ Fe吸氧腐蚀：Fe->Fe2+,O2->OH-，Fe2++OH-->Fe(OH)2->Fe(OH)3->Fe2O3·nH2O
+ 高炉炼铁需加入CaCO3（高温下分解为CaO和CO2,CaO脱硫，CO2和C反应为CO用于还原氧化铁），炉渣为SiO2和CaSiO3
## Cu
+ Cu受热分解：Cu(OH)2->CuO-高温->Cu2O-高温->Cu
+ 热还原法：H2+CuO=加热=Cu+H2O
+ 湿法炼铜：Fe+CuSO4==Cu+FeSO4
+ Cu在潮湿空气中最终生成碱式碳酸铜（铜绿）Cu2(OH)2CO3（化学方程式：2Cu+O2+H2O+CO2==Cu2(OH)2CO3）
+ 区分Cu和Cu2O:Cu2O可溶于H2SO4(歧化反应生成Cu和CuSO4,溶液变蓝)
+ 常常使用灼热的铜网除去N2中的O2,用CuO除去CO2中的CO
+ FeCl3腐蚀铜板::2Fe3++Cu＝2Fe2++Cu2+
## Si
+ C还原法：SiO2+2C=高温=Si+2CO↑但与Si和C的还原性无关(实际上还原性Si>C)
+ 高温下Na2CO3+SiO2=高温=Na2SiO3+CO2↑，但与H2CO3和H2SiO3的酸性大小无关(实际上酸性H2CO3>H2SiO3)
+ SiO2+4HF==SiF4↑+2H2O
+ Si+4HF==SiF4↑+2H2↑
+ 与碱、碱性氧化物反应：SiO2+CaO=高温=CaSiO3、SiO2+2NaOH==Na2SiO3+H2O
+ 仅硅与碱反应放出氢气（其余非金属单质反应为歧化反应）：Si+2NaOH+H2O==Na2SiO3+2H2↑
+ 制备高纯硅：Si+Cl2-加热->SiCl4+H2-加热->Si+HCl
+ 玻璃：Na2CO3、CaCO3、SiO2共热变为Na2SiO3、CaSiO3、SiO2混合物，没有固定熔点
+ 有色玻璃加入金属氧化物，钢化玻璃通过加热普通玻璃后急速冷却制造
+ 水泥：用CaCO3和粘土制造,成分为2CaO·SiO2、3CaO·SiO2、3CaO·Al2O3，具有水硬性
+ 沙子、水泥、碎石混合物称为混凝土，混凝土一般以钢筋做结构
## Br
+ 氧化性：Cl2>Br2>I2，还原性I->Fe2+>Br->Cl-
+ 海水提溴：苦卤（高浓度Br-）-Cl2氧化->Br2->鼓入热空气或水蒸气使其挥发
+ AgBr见光易分解做感光材料，NaBr/KBr做镇静剂，Br2做燃料抗爆剂
## S
+ 工业制取浓硫酸SO2-常压、高温、催化剂->SO3->H2SO4（用浓硫酸吸收SO3），常压的原因：常压下SO2转化率已很高，再增大压强转化率提高不大但生产成本增加
+ 工业生产时加入CaO可起到烟气固硫的作用（CaSO4)
+ SO2的分子结构为V形而不是直线形
+ 火药：S、KNO3(硝石)、C
+ SO2+H2S==3S↓+2H2O（S2-往往与其它含S离子/气体歧化生成S单质）
## N
+ NH3催化氧化4NH3+5O2=(催化剂,加热)=4NO+6H2O
+ NH3为三角锥形分子
+ Mg3N2：N2和Mg点燃生成，Mg3N2和过量HCl反应生成两种盐：MgCl2+NH4Cl
+ NO2溶于NaOH歧化:2NO2 + 2NaOH = NaNO2 + NaNO3 +H2O
+ NO2、NO与NaOH反应：NO + NO2 + 2NaOH = 2NaNO2 +H2O
+ 氮气由于密度与空气相近不能用排空气法收集
+ 铵态氮肥包括(NH4)2SO4、NH4HCO3、NH4Cl不能与碱性物质（草木灰）共同施用，储存碳酸氢铵时应密封并放在阴凉通风处，施肥时应埋在土下保持肥效。
+ 实验室制氨气：2NH4Cl+Ca(OH)2=加热=2NH3↑+CaCl2+2H2O（用向下排空气法收集，用湿润的红色石蕊试纸检验是否收集满，收集试管口要塞棉花防止空气与NH3对流，加热试管要朝下）
+ HNO3不稳定，受热或见光分解生成NO2、O2、H2O
+ 实验室制NO2: Cu+4HNO3(浓)===Cu(NO3)2+2NO2↑+2H2O，向上排空气法收集
+ 氮的常见氧化物：NO、N2O、NO2、N2O3、N2O4、N2O5
## Mg
+ 海水提镁：CaCO3->CaO->Ca(OH)2-海水中的Mg2+->Mg(OH)2-HCl->MgCl2溶液-浓缩、干燥->MgCl2-通电->Mg,Cl2->回收制HCl
+ 2Mg+CO2=点燃=2MgO+C（特殊置换反应，发出耀眼白光，实验时要打磨表面氧化膜并在底层铺细沙或水）
+ Mg密度小，强度高，机械性能好，称为国防金属。镁常用做还原剂和脱氧剂
## C
+ 草酸分解生成CO、CO2、H2O
+ 制水煤气C＋H2O(g)==高温==CO＋H2
### 侯氏制碱法
1. NH3＋CO2＋H2O=NH4HCO3
2. NH4HCO3＋NaCl=NaHCO3↓＋NH4Cl
3. 2NaHCO3 ==加热==Na2CO3＋H2O＋CO2↑
+ 制的碱指Na2CO3
+ 主要利用NaHCO3在溶液中溶解度较小，所以先制得NaHCO3，再利用碳酸氢钠不稳定性分解得到纯碱
+ 第二步中NaHCO3溶解度最小所以从饱和溶液中析出
+ 优点：革除了CaCO3制CO2这一工序 第三步的CO2可循环使用 第二步造氮肥NH4Cl
# @仪器的使用
+ 读数精度：天平0.1g，pH试纸到整数
+ 注意量筒不能润洗：因为量筒的设计之初就考虑到了量筒的挂壁，这部分体积是被考虑进去的，倒出去的体积就是我们需要的体积。
## 锥形瓶
+ 滴定时不能用待测液润洗
+ 注入的液体不超其容积的1/2，过多易造成喷溅。
+ 加热时使用石棉网（电炉加热除外）。
## 蒸馏烧瓶
+ 蒸馏时温度计水银球要与支管口相平，加热时要垫石棉网，烧瓶内应加入碎瓷片/沸石防液体暴沸
+ 盛放液体应占总体积1/3到2/3
+ 冷却水由下口进上口出，蒸馏前的操作是通冷却水
## 蒸发皿
+ 能耐高温，加热后不能骤冷，防止破裂。
+ 应使用坩埚钳取放蒸发皿，加热时用三脚架或铁架台固定。
+ 液体量多时可直接加热 ，量少或黏稠液体要垫石棉网或放在泥三角上加热。
+ 加热蒸发皿时要不断的用玻璃棒搅拌，防止液体局部受热四处飞溅。
+ 大量固体析出后就熄灭酒精灯，用余热蒸干剩下的水分。
+ 加热时，应先用小火预热，再用大火加强热。
+ 要使用预热过的坩埚钳取热的蒸发皿。
+ 用蒸发皿盛装液体时，其液体量不能超过其容积的三分之二。
## 试管
+ 装溶液时不超过试管容量的1/2，加热时不超过试管容量的1/3。
+ 用滴管往试管内滴加液体时应悬空滴加，不得伸入试管口。
+ 取块状固体要用镊子夹取放至试管口，然后慢慢竖起试管使固体滑入试管底，不能使固体直接坠入，防止试管底破裂。
+ 加热使用试管夹。加热盛有固体的试管时，管口稍向下，加热液体时倾斜约45°。
+ 受热要均匀，以免暴沸或试管炸裂。
+ 加热后不能骤冷，不能在试管未冷却至室温时就洗涤试管，防止破裂。
+ 加热时要预热，防止试管骤热而爆裂。
+ 加热时要保持试管外壁没有水珠，防止受热不均匀而爆裂。 [1]
+ 使用试管夹夹取试管时，将试管夹从试管的底部往上套，夹在试管中上部三分之一的部位。 [1]
+ 加热应用外焰。
## 滴定管
+ 滴定管精度0.01mL，0刻度在上端
+ 使用滴定管的第一步是查漏
### 滴定管的选用
+ 酸式滴定管下部为玻璃，碱式滴定管下部为橡胶
+ 需避光的滴定液（如硝酸银滴定液、碘滴定液、高锰酸钾滴定液、亚硝酸钠滴定液、溴滴定液等），需用棕色滴定管。
+ 具有氧化性的溶液或其他易与橡皮起作用的溶液，如高锰酸钾、碘、硝酸银等不能使用碱式滴定管。
## 容量瓶
+ 注意用容量瓶配溶液时要冷却后倒入
+ 少量蒸馏水无影响
+ 写名称时记得带容量
+ 溶液转移时玻璃棒应伸入刻度线下，定容时胶头滴管在瓶口
## 坩埚
+ NaOH、Na2CO3等与SiO2反应的物质不能在瓷坩埚中加热
+ 可直接受热，加热后不能骤冷，用坩埚钳取下
+ 坩埚受热时放在泥三角上
## 启普发生器
+ 不加热，块状固体与液体反应制气体
+ 过氧化氢与二氧化锰制氧气不能用启普发生器（泡沫冲入导管且反应释放大量的热）
