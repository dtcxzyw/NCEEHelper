# Li
+ 最轻的金属
+ 硬度和熔点最高的碱金属
+ 焰色反应紫红色
+ LiOH溶解度小
+ 工业制取：电解LiCl
+ 在空气中形成氧化层
# Be
+ 轻金属
+ 最硬的碱土金属
+ 有两性
+ 在空气中形成氧化层
+ 倾向于形成共价键
+ BeO、Be(OH)2有两性
+ Be(OH)2不易溶于水，溶于酸形成Be2+，溶于碱形成[Be(OH)4]2-
+ BeCl2是共价型化合物，在空气中会吸潮并水解为BeO
# B
+ 硼砂（Na₂B₄O₇·10H₂O）
+ 在自然界中主要以硼酸和硼酸盐的形式存在
+ 硼酸，化学式H₃BO₃，水溶液呈弱酸性
+ B能在空气中燃烧生成B₂O₃和少量BN，在室温下即能与F₂发生反应
+ B能从许多稳定的氧化物（如SiO₂，P₂O₅，H₂O等）中夺取氧而用作还原剂。
+ 硼不与盐酸作用，但与热浓H₂SO₄，热浓HNO₃作用生成硼酸
+ 在氧化剂存在下，硼和强碱共熔得到偏硼酸盐：2B+2NaOH+3KNO₃==2NaBO₂+3KNO₂+H₂O
# C
+ 碳是生铁、熟铁和钢的成分之一。
+ 碳作为还原剂拥有和氢气、一氧化碳相似的化学性质（但生成物不同），都可以从金属氧化物中还原出金属单质。
+ 有机物的性质与无机物大不相同，它们一般可燃(氧化反应)、不易溶于水
# N
+ 氮是空气中最多的元素
+ 硝石(KNO3)
+ 自然界中绝大部分的氮是以单质分子氮气的形式存在,氮的最重要的矿物是硝酸盐。
+ 主要用途：用于合成氨，制硝酸，用作物质保护剂，冷冻剂。
+ 分馏液态空气可获得氮气。
+ N₂(g)+3Mg(s)→Mg₃N(s)
+ 氮气转化为氮的化合物的方法叫做氮的固定
# O
+ 氧是地壳中最丰富、分布最广的元素
+ 单质氧在大气中占20.9%。
+ 但是接近地表的臭氧则是一种污染，这些臭氧主要存在于光化学烟雾中。
+ 氧的化合价很特殊一般为-2价和0价。而氧在过氧化物中通常为-1价。在超氧化物中为-1/2，臭氧化物中氧为-1/3
+ 黄金外的所有金属都能和氧发生反应生成金属氧化物
+ 实验室制取：KMnO4加热、H2O2+MnO2、KClO3+MnO2加热
+ 工业制取：分离液态空气法;电解制氧法;分子筛制氧法（吸附法）
+ 臭氧是一种有鱼腥臭味的蓝色气体。
+ 氧气含量过高时也会发生氧中毒
# F
+ F2一种淡黄色剧毒的气体
+ 氟是特种塑料、橡胶和冷冻机（氟氯烷）中的关键元素。
+ 氟气的腐蚀性很强，化学性质极为活泼，是氧化性最强的物质之一
+ 用于制金属冶炼中的助熔剂（冰晶石等）、合成氟利昂等冷却剂、含氟塑料和含氟橡胶、添加于牙膏中
+ 人体含氟主要分布在骨骼、牙齿中
# Na
+ 钠原子的最外层只有1个电子，很容易失去，所以有强还原性
+ 工业制取：电解NaCl或NaOH
+ 浸放于液体石蜡、矿物油和苯系物中密封保存
# Mg
+ 镁能和二氧化碳发生燃烧反应，因此镁燃烧不能用二氧化碳灭火器灭火。
+ 一些烟花和照明弹里都含有镁粉，就是利用了镁在空气中燃烧能发出耀眼的白光的性质。
+ Mg+2H2O（热水）== Mg(OH)₂+H₂↑
+ 2Mg+CO₂=点燃=2MgO+C
+ 工业上利用电解熔融氯化镁或在电炉中用硅铁等使其还原而制得金属镁
+ 氯化镁可以从海水中提取
# Al
+ 铝元素在地壳中的含量仅次于氧和硅，居第三位，是地壳中含量最丰富的金属元素。
+ 铝制品在一类致癌物清单中。
+ 明矾KAl(SO4)2·12H2O
+ 黏土（Al2O3·2SiO2·2H2O）制成陶器
+ 工业制取：将氧化铝溶解在冰晶石（Na3AlF6）中电解
+ 自然界存在的刚玉即属于α一Al₂O₃，它的硬度仅次于金刚石
+ Al₂O₃由于含有不同的杂质而有多种颜色。例如含微量Cr（III）的呈红色，称为红宝石；含有Fe（II），Fe（III）或Ti（IV）的称为蓝宝石。
+ 不电解熔融AlCl₃炼Al；原因：AlCl₃是共价化合物，其熔融态导电性极差。
+ 铝是活泼金属，在干燥空气中铝的表面立即形成厚约50埃（1埃=0.1纳米）的致密氧化膜，使铝不会进一步氧化并能耐水；但铝的粉末与空气混合则极易燃烧；熔融的铝能与水猛烈相应的金属；铝是两性的，极易溶于强碱，也能溶于稀酸。
+ 铝热反应 2Al + Fe₂O₃==点燃== Al₂O₃+2Fe（铝热反应） 8Al + 3Fe₃O4==高温== 4Al₂O₃+9Fe
+ 铝与沸水几乎没有反应现象，不过铝在加热条件下就可以与水蒸汽发生明显反应，但反应一开始就与水中的氧气生成致密氧化膜阻止反应进一步进行。
+ 铝箔广泛用于包装香烟、糖果等
# Si
+ 在地壳中，它是第二丰富的元素
+ 燧石 （硅质岩石）SiO2
+ 石英、水晶等是纯硅石的变体。矿石和岩石中的硅氧化合物统称硅酸盐，长石、云母、黏土、橄榄石、角闪石等等都是硅酸盐类；水晶、玛瑙、碧石、蛋白石、石英、砂子以及燧石等等都是硅石。土壤、黏土和砂子是天然硅酸盐岩石风化后的产物。
+ 有无定形硅和晶体硅两种同素异形体。晶体硅为灰黑色，无定形硅为黑色
+ 硅有明显的非金属特性，可以溶于碱金属氢氧化物溶液中，产生（偏）硅酸盐和氢气。
+ Si + O₂ == SiO₂，条件：加热 Si + 2F₂ == SiF₄ Si + 2Cl₂ == SiCl₄，条件：高温
+ 2MgO + Si=高温真空 =Mg(g)+SiO₂（硅热还原法炼镁）
+ 只与氢氟酸反应：Si + 4HF == SiF₄↑ + 2H₂↑
+ 工业上生产硅是在电弧炉中还原硅石（SiO2含量大于99%）。使用的还原剂为石油焦和木炭等。
# P
+ 白磷在高于44摄氏度时会在氧气中自燃
+ 磷的正常机能需要维生素D（维生素食品）和钙（钙食品）来维持。
+ 白磷或黄磷是无色或淡黄色的透明结晶固体
+ 红磷是红棕色粉末，无毒
+ 在自然界中，磷以磷酸盐的形式存在
+ 白磷在空气中自燃生成氧化物。P4+3O2==P4O6（化合反应）
+ 白磷与卤素单质剧烈反应，在氯气中也能自燃生成三氯化磷和五氯化磷。P4+6Cl2==4PCl3（化合反应）P4+10Cl2==4PCl5（化合反应）
+ 白磷能被硝酸氧化成磷酸。3P+5HNO3+2H2O==3H3PO4+5NO↑（氧化还原反应）
+ 白磷溶解在热的浓碱中，歧化生成磷化氢和次磷酸盐。P4 +3OH- +3H2O == PH3+ 3H2PO2-
# S
+ 通常单质硫是黄色的晶体，又称作硫磺。
+ 硫燃烧时有消毒和漂白的作用。
+ 单质硫主要存在于火山周围的地域中。 以化合态存在的硫多为矿物
+ 石膏（CaSO4·2H2O）、芒硝（Na2SO4·10H2O）、重晶石（BaSO4）、矾石[(AlO)2SO4·9H2O]、明矾石[K2SO4·Al2(SO4)3·24H2O]
+ 硫和汞常温下即可反应 ，故可用硫处理泄露的汞
+ 硫可与变价金属反应生成低价态金属硫化物
+ 二氧化硫有漂白性，但只能漂白品红等极少数物质，漂白原理是与有机色素化合生成无色物质，且漂白有暂时性，因为化合物会分解。
+ 二氧化硫可被大部分氧化剂（包括氧气）氧化，二氧化硫使高锰酸钾溶液变无色不是漂白。
+ 浓硫酸主要有四个性质：难挥发性、吸水性、脱水性、强氧化性。前两个是物理性质，后两个是化学性质。
+ 利用难挥发性用浓硫酸制其他强酸（通常是氯化氢）
+ 往氢硫酸中加入亚硫酸（或通入二氧化硫），可以制取硫。
# Cl
+ 氯气微溶于水，易溶于碱液，易溶于四氯化碳、二硫化碳等有机溶剂。
+ 氯气遇水歧化为盐酸和次氯酸，次氯酸不稳定易分解放出游离氧，其中次氯酸具有漂白性
# K
+ 钾在自然界没有单质形态存在，钾元素以盐的形式广泛的分布于陆地和海洋中
+ 钾盐以硝石（硝酸钾，KNO3），明矾（十二水合硫酸铝钾，KAl(SO4)2·12H2O），还有草木灰（碳酸钾，K2CO3）的形式
+ 生产金属钠用的是电解法，但在生产金属钾中却不能用此法，因为钾太容易溶解在熔化的KCl中，无法分离收集。金属钾的生产方法采用金属钠与氯化钾在高温下的反应：Na+KCl=K↑+NaCl。
+ 一般把钾、钠储存在煤油和液体石蜡中。
# Ca
+ 主要的含钙矿物有石灰石CaCO3、白云石CaCO3·MgCO3、石膏CaSO4·2H2O、萤石CaF2、磷灰石Ca5（PO4）3F等。蛋壳、珍珠、珊瑚、一些动物的壳体和土壤中都含有钙。
+ 化学性质活泼，在空气中表面上能形成一层氧化物或氮化物薄膜，可减缓进一步腐蚀。
+ 钙跟碳在高温下反应生成碳化钙CaC2。
+ 电解无水氯化钙制钙
# Ti
+ 二氧化钛是世界上最白的东西用作颜料
+ 钛属于稀有金属，实际上钛并不稀有，其在地壳中的丰度占第七位，占0.45%，远远高于许多常见的金属。但由于钛的性质活泼，对冶炼工艺要求高，使得人们长期无法制得大量的钛，从而被归类为“稀有”的金属。用于冶炼钛的矿物主要有钛铁矿（FeTiO3）、金红石（TiO2）和钙钛矿等。矿石经处理得到易挥发的四氯化钛，再用镁还原而制得纯钛。
